## Setup (once per clone)

```bash
./scripts/setup-git-hooks.sh
```

This enables `.githooks/prepare-commit-msg`, which appends both lines to every new commit.

## Pair Extraordinaire

GitHub counts co-authored commits on **merged pull requests**. Prefer feature branch → PR → merge over direct pushes to `main` when you want achievement credit.
