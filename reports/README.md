# Reports

## Active pipeline

Portfolio and worksheet validation reports live under `reports/implementation/` when generated by phase scripts.

The **simple LO rubric** workflow uses:

- `python run_portfolio_builder.py <student_id>` → `students/<id>/portfolio.json`
- `python validate_pipeline_outputs.py` / `python validate_schemas.py`

## Archived ECD milestones (M1–M5)

The multi-layer Evidence-Centered Design chain (Observable Behaviours → ILO →
Domain → AI-CFT policy) and its milestone validators/summaries were moved to
`archive/ecd_v1/reports/` and `archive/ecd_v1/scripts/`. They are **not** part of
the active pipeline. See `archive/ecd_v1/README.md`.

## Policy

- No `freeze` metadata on active pipeline artifacts.
- Human-readable summaries are optional; machine validation uses root-level validators.
